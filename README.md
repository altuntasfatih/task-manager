# task-manager
It is a poc task-manager api. It runs on embedded memory(BadgerDb)

 * Run with `go run cmd/main.go `
 * Swagger url `http://127.0.0.1:8080/swagger/index.html`

## To test 
 * Test  with `go test ./...`
